# Project Architecture (summary)

This repository contains a Vite + React frontend and a Python-based route optimization backend (optimal-route-kl).

## Top-level:
*   `optimal-route-kl/`: The main project directory.
    *   `frontend/`: The React frontend.
    *   `backend/`: The Python backend.
*   `.github/`: GitHub Actions workflows.
*   `firebase.json`: Firebase configuration.
*   `.firebaserc`: Firebase project configuration.
*   `GEMINI.md`: AI development guidelines.
*   `README.md`: Project documentation.
*   `.gitignore`: Files and directories to be ignored by git.
*   `.env.example`: Example environment variables.
*   `SECURITY_CHECKS.md`: Security checklist.
*   `ARCHITECTURE.md`: This file.
